## 概要

プロジェクトの依存パッケージ導入、環境変数設定、Supabase プロジェクトの準備。

## 実装メモ

### インストール

```bash
npm install @supabase/supabase-js @supabase/ssr server-only
```

### 環境変数（`.env.local`）

```env
NEXT_PUBLIC_SUPABASE_URL=https://<project-ref>.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=<anon-key>
SUPABASE_SERVICE_ROLE_KEY=<service-role-key>
ADMIN_EMAIL=your-admin@example.com
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

### ファイル構成

```
lib/
  supabase/
    client.ts   # createBrowserClient（Client Component 用）
    server.ts   # createServerClient + cookies()（Server 用）
proxy.ts        # セッションリフレッシュ（プロジェクトルート）
```
